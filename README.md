# Avanti Org Charts

Employee reporting structure for Avanti Fellows, served as a static GitHub Pages site.

**Live**: https://asaxena.github.io/AvantiOrg/

The page renders an indented-row tree from `org_data.json`. Source workbook lives in the `Employee Org Charts` working directory; build scripts there regenerate `index.html` + `org_data.json` for republishing.
